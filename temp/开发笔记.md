从模板初始化插件项目
dotnet new bepinex5plugin -n <项目名> -T <TFM> -U <Unity>
dotnet new bepinex5plugin -n HelloWorldMod -T netstandard2.1 -U 2022.3.62.7762112

white knuckle 作弊指令
cheats 开启/关闭作弊
deathgoo-height -Infinity Mass高度无限低
listentityprefabs 全部实体列表
spawnentity <实体名> 生成实体
spawnentity item_artifact_remote 生成远程文物
spawnentity denizen_bloodbug_worker 生成血虫
getgenerationseed 显示当前种子
setgenerationseed <种子> 加载指定种子 

在启动游戏前设置

set COMPlus_ZapDisable=1
set COMPlus_ReadyToRun=0
